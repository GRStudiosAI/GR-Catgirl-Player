# 🎵 GR Catgirl Player

![GR Catgirl Player Banner](assets/backgrounds/banner.png)

GR Catgirl Player is a custom media player developed by GR Studios featuring a cyberpunk-inspired interface, GR Catgirl branding, and support for images, GIFs, videos, and audio.

Designed to provide a unique media experience while remaining lightweight, customizable, and easy to use.

---

## ✨ Features

### 🎞 GIF Player

* Animated GIF playback
* Loop support
* Replay support
* GIF viewing tools

### 🎥 Video Player

* MP4 support
* WEBM support
* MKV support (Electron Version)
* AVI support (Electron Version)
* MOV support (Electron Version)

### 🎵 Music Player

* MP3 playback
* WAV playback
* OGG playback
* FLAC playback (Electron Version)
* M4A playback (Electron Version)

### 🖼 Image Viewer

* PNG support
* JPG support
* WEBP support
* SVG support
* Fullscreen viewing

### 🎮 Custom Player Controls

* Play
* Pause
* Stop
* Replay
* Rewind
* Fast Forward
* Previous
* Next
* Volume Control
* Mute
* Fullscreen
* Timeline Seeking

## 🚀 Running The HTML Version

Open:

```text
index.html
```

in your browser.

No installation required.

---

## ⚡ Running The Electron Version

Install dependencies:

```bash
npm install
```

Start Electron:

```bash
npm run electron
```

Development mode:

```bash
npm run electron:dev
```

Build executable:

```bash
npm run electron:build
```
